# PDF 智能解析工具 — 使用说明

## 这是什么？

一个能把 PDF 文件中的文字提取出来的桌面工具。
- **文字型 PDF**（可直接复制的）→ 自动提取文字
- **扫描型 PDF**（每页是图片）→ 自动 OCR 识别文字

支持导出 Markdown、TXT、JSON 格式。

## 界面操作说明

### 左侧 — 操作面板

1. **选择文件** → 点击「浏览」选择 PDF
2. **选择解析方式**（默认自动检测）：
   - `自动检测` → 有文字直接提取，没文字自动 OCR
   - `文字提取` → 强制文字模式
   - `OCR 识别` → 强制 OCR
3. **设置 DPI** → 数字越大越清晰越慢（默认 300）
4. **设置页数** → 0=全部，可只解析前几页测试
5. **点击「开始解析」**

### 右侧 — 预览

- 解析完后用 ◀ ▶ 翻页查看

### 导出

- `导出 MD` → Markdown 格式
- `导出 TXT` → 纯文本
- `导出 JSON` → 结构化数据

## 常见问题

### OCR 是什么？
把图片里的文字"读"出来的技术。扫描版 PDF 需要 OCR。

### OCR 速度？
每页约 2-5 秒（300 DPI），300 页约 10-20 分钟。

### 识别准确率？
中文扫描件通常 95% 以上。要求更高可调高 DPI 或导出后手动修正。
